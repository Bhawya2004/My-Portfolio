# Bhawya Gulati | Personal Portfolio

A modern, responsive, and highly interactive personal portfolio website showcasing my skills, projects, certificates, and achievements in Software Engineering, Artificial Intelligence, and Machine Learning. The portfolio is designed with a focus on UI/UX, featuring smooth animations, a custom project timeline, and an integrated AI-themed Resume Chatbot.

## ✨ Features

- **Resume Chatbot (Smart Assistant):** A floating, glassmorphic chatbot widget built with Vanilla JavaScript that answers questions about my resume, skills, projects, education, and contact information. Includes a typing indicator and smart keyword-based responses.
- **Interactive Project Timeline:** A custom checkpoint-style vertical scrolling timeline where connector lines grow and project cards animate into view dynamically as you scroll down the page.
- **3D Certificate Flip Cards:** Interactive cards for displaying 9+ certifications (including Generative AI, Prompt Engineering, ML, and Networking) with detailed descriptions on the back.
- **Glassmorphic UI & Animations:** Modern design aesthetics, soft shadows, custom bouncing spring animations, and smooth reveals.
- **Responsive Design:** Fully optimized across mobile, tablet, and desktop devices.
- **Dark/Light Mode:** Seamless theme switching with custom warm peach and dark color palettes.
- **Integrated Resume Preview:** A built-in modal to view and download my complete curriculum vitae (CV) directly from the site without navigating away.

## 🛠️ Technology Stack

- **Frontend Structure:** HTML5 (Semantic HTML for accessibility)
- **Styling:** CSS3 (Custom properties/variables, Flexbox/Grid layouts, custom keyframe animations, and media queries). **No external CSS frameworks used.**
- **Interactivity:** Vanilla JavaScript (ES6+), DOM manipulation, scroll event handling (Intersection Observer), and custom chatbot logic.
- **Icons:** FontAwesome
- **Fonts:** Google Fonts (Poppins & Inter)

## 🚀 Running the Project Locally

To run this project on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Bhawya2004/My-Portfolio.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd My-Portfolio
   ```
3. **Run a local server:**
   Since this is a static site without a build step, you can simply open `index.html` in your browser. However, for the best experience (and to avoid CORS issues if loading local fonts/assets), run a simple local web server:
   - **Using Python 3:** `python3 -m http.server 8000`
   - **Using Node.js:** `npx serve`
   - Or use the **Live Server** extension in VS Code.
4. Open your web browser and visit `http://localhost:8000`.
